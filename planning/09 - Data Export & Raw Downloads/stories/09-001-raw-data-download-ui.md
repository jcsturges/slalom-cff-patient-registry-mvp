# Raw Data Download UI

**Story ID:** 09-001
**Epic:** 09 - Data Export & Raw Downloads
**Priority:** P1
**SRS Reference:** Section 8.1

## User Story
As a care program user, I want to select eCRFs, date ranges, and filters for a raw data download so that I can export registry data for offline analysis.

## Description
I can export registry data for offline analysis

## Dependencies
- 06-001, 01-004

## Acceptance Criteria
- [ ] 'Raw Data Download' control panel accessible from main menu
- [ ] Select one or more eCRFs or 'All Forms'
- [ ] Date range filter (from/to dates)
- [ ] Completeness filter: 'Complete records only' or 'All records'
- [ ] Diagnosis filter: CF vs CFTR-related vs CRMS/CFSPID
- [ ] Program-level data access enforced (only user's program patients)
- [ ] Brief descriptions for each control/filter

## Technical Notes
- Control panel should be intuitive with clear labels and tooltips for each option.
