# Incomplete Records Report

**Story ID:** 08-007
**Epic:** 08 - Reporting & Report Builder
**Priority:** P0
**SRS Reference:** Section 7.4.1

## User Story
As a care program user, I want to view a list of incomplete unlocked forms for my program so that I can track data entry progress for the current reporting year.

## Description
I can track data entry progress for the current reporting year

## Dependencies
- 06-002

## Acceptance Criteria
- [ ] Report shows incomplete unlocked forms for selected reporting year
- [ ] Columns: CFF ID, First Name, Last Name, Incomplete form type, Incomplete sub-form status
- [ ] One row per incomplete form instance
- [ ] Default sort by CFF ID, forms in descending date order
- [ ] User can select which reporting year to view
