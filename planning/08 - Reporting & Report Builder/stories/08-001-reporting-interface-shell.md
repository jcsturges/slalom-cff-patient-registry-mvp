# Reporting Interface Shell

**Story ID:** 08-001
**Epic:** 08 - Reporting & Report Builder
**Priority:** P0
**SRS Reference:** Section 7.1

## User Story
As a user, I want to access a Reporting section from the main menu with all available reports listed by title so that I can find and run any report I have access to.

## Description
I can find and run any report I have access to

## Dependencies
- 03-003, 03-010

## Acceptance Criteria
- [ ] Reporting accessible from main menu
- [ ] Reports listed by title
- [ ] CP users see: Report Builder, Incomplete Records, Patients Due Visit, Diabetes Testing
- [ ] Foundation Admins see all reports including administrative and audit reports
- [ ] Each report has a title and description at the top

## Technical Notes
- Central reporting landing page with categorized report list.
